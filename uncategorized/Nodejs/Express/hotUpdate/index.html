Hot Update

hotUpdate in Express

nodemon

Nodemon 是一个在开发过程中自动监视文件更改并重新启动 Node.js 应用程序的工具。

  1. Install
  • npm install -g nodemon
  1. Usage
  • nodemon app.js (修改express启动方式由node改为nodemon)
