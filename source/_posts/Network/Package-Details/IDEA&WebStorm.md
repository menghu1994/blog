## Hot key
+ ctrl + i
 - 实现接口方法
+ ctrl + j
 - 自动代码片段
+ ctrl + h
 - 查看当前类的子类
+ ctrl + alt + u
 - 显示类之间的关系
+ ctrl + alt + o
 - 删除类中没有用到的package
+ ctrl + f12
 - 找到controler,定位代码位置
+ ctrl + ~
 - 常用主题,显示模式,快捷键配置
+ ctrl + alt + L 
 - 格式化代码
+ ctrl + w
 - 选中并扩大当前选择(其他软件为关闭当前页)
+ alt + arrow
 - 切换窗口
+ ctrl + shift + i
 - 展开方法提示
+ alt + ~
 - git 快捷操作

+ ctrl + alt + v 重构变量
