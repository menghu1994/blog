Frp实现内网穿透

frp 是一个反向代理应用，客户端（frpc）主动连接公网服务器（frps）并建立持久连接，从而让外网用户通过 frps 访问位于内网的 frpc 机器上的服务。

通过sftp将本地文件上传到服务器

创建frps.ini文件

  • web dashboard 面板端口为 7500
  • 可通过 http://x.x.x.x:7500 登录面板查看连接状态
